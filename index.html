¡Listo para Publicar!

Tu Agente de IA ya se ha encargado del código y las rutas. Ahora solo debes seguir estos pasos para subir tu web y tenerla online gratis.

Progreso General 0%

Requisitos Previos

  • Cuenta activa en GitHub.
  • Archivos de la web preparados.
¡Copiado al portapapeles!
Paso 1

Crear un Nuevo Repositorio en GitHub

  1. 1 Inicia sesión en tu cuenta en github.com .
  2. 2 Haz clic en el botón de suma + arriba a la derecha y selecciona New repository.
  3. 3 Escribe un nombre elegante para tu proyecto (ej: mi-sitio-web).
  4. 4 Asegúrate de marcarlo como Public (es indispensable para usar la opción gratuita).
  5. 5 Deja todas las demás opciones desmarcadas y haz clic en Create repository.
Paso 2

Subir los archivos del proyecto

  1. 1 En la página inicial de tu nuevo repositorio, busca y pulsa el enlace que dice "uploading an existing file".
  2. 2 Selecciona y arrastra todos los archivos de tu web dentro del área gris de la pantalla.
  3. 3 Abajo del todo, escribe un título descriptivo en el campo Commit (ej. Subida inicial).
  4. 4 Presiona el botón verde Commit changes y espera a que se procese.
Paso 3

Activar el Despliegue de Pages

  1. 1 Ve a la pestaña Settings (Configuración) en el menú superior del repositorio.
  2. 2 En el panel izquierdo busca el menú de Pages.
  3. 3 Bajo la sección *Build and deployment*, cambia la rama (Branch) de None a main (o master).
  4. 4 Guarda los cambios haciendo clic en Save.
Paso 1

Inicializar tu repositorio local

Abre tu terminal dentro de la carpeta del proyecto y ejecuta estos comandos secuenciales:

git init
git checkout -b main
git add .
git commit -m "Commit inicial"
Paso 2

Vincular con el servidor y subir

Crea el repositorio vacío en la web de GitHub, copia su URL HTTPS y pégala en los siguientes comandos:

# Añade la dirección de tu repo de GitHub
git remote add origin https://github.com/tu-usuario/nombre-repo.git

# Empuja los archivos hacia la nube
git push -u origin main
Paso 3

Actualizaciones automáticas futuras

Para realizar nuevas actualizaciones del proyecto, solo ejecuta en consola:

git add .
git commit -m "Actualización de contenido"
git push origin main

¿Dónde estará visible tu página?

Cuando el despliegue finalice, podrás acceder a tu sitio usando el enlace de tu cuenta:

https://tu-usuario.github.io/tu-repositorio/

Nota de despliegue: El proceso inicial toma habitualmente entre 1 y 5 minutos.
